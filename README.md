# Analise-logistica-sp-excel-sql
Projeto de análise de dados de entregas em São Paulo(Higienópolis,Jardins,Centro, Paulista) usando SQL para estruturação e Excel para Dashboard.

🚚 Logística Inteligente: Dashboard de Entregas SP Capital
Este projeto apresenta uma solução de monitoramento para operações de entrega em bairros de alta densidade na cidade de São Paulo (Higienópolis, Jardins, Paulista e Centro). O objetivo é transformar dados operacionais brutos em indicadores de performance (KPIs) para otimização de rotas e prazos.

🛠️ Tecnologias e Metodologia
SQL (SQLite/PostgreSQL): Modelagem de dados relacional. Criação de tabelas de Pedidos e Entregadores com relacionamentos (JOINs) para análise de faturamento por colaborador.
Excel Avançado: Processamento de 50+ registros reais, utilizando fórmulas de busca e referência (ÍNDICE/CORRESP) e funções estatísticas condicionais (SOMASES, CONT.SES).
Business Intelligence: Construção de Dashboard Dinâmico focado em:
Ticket Médio por Bairro: Identificação de regiões com maior rentabilidade.
SLA (Service Level Agreement): Monitoramento de taxa de atrasos e entregas pendentes.
